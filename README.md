# 🦸‍♂️ Classificador de Nível de Herói

Projeto desenvolvido como desafio da DIO com o objetivo de praticar lógica de programação utilizando C#.

## 🚀 Tecnologias utilizadas

- C#
- .NET
- Console Application

## 📚 Conceitos aplicados

- Variáveis
- Operadores
- Estruturas de decisão (switch)
- Laços de repetição (while)
- Validação de entrada de dados

## 🎯 Objetivo

Classificar o nível de um herói com base na sua quantidade de experiência (XP).

### Regras de classificação:

| XP                | Nível       |
|------------------|------------|
| < 1000           | Ferro      |
| 1000 - 2000      | Bronze     |
| 2001 - 5000      | Prata      |
| 5001 - 7000      | Ouro       |
| 7001 - 8000      | Platina    |
| 8001 - 9000      | Ascendente |
| 9001 - 10000     | Imortal    |
| >= 10001         | Radiante   |


## ⚡ Melhorias implementadas

- Validação de entrada de dados (evita erros)
- Loop para múltiplas execuções
- Tratamento de nome vazio
